# 🕯️ Ég a Gyertya - Projekt Dokumentáció

## 🎯 Vízió
Egy prémium megjelenésű, természetbarát, kézműves szójaviasz gyertyákat és dekorációkat bemutató webshop készítése. 
**Cél:** A morelightcandle.hu-nál modernebb, kifinomultabb és jobb felhasználói élményt nyújtó oldal.

## 🎨 Design & Arculat ("Föld és Levendula")
- **Főszín:** Levendula lila (#8A6FDF) - elegancia és nyugalom.
- **Háttér:** Törtfehér/Viasz szín (#FAF9F6) - természetesség.
- **Szöveg:** Mélyszürke/Földszínek (#374151, #78716C) - olvashatóság.
- **Stílus:** Letisztult, szellős, modern tipográfia, finom animációk.

## 🛠️ Technológiai Stack
- **Frontend:** React (Vite)
- **Styling:** Tailwind CSS (egyedi konfigurációval)
- **Ikonok:** Lucide React
- **Animációk:** Framer Motion
- **Navigáció:** React Router

## 📈 Aktuális Állapot (2026.01.06)
